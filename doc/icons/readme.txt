PWA用のアイコンファイルについて

このディレクトリには以下のアイコンが必要です：
- icon-192x192.png - 192x192ピクセルサイズのPNGアイコン
- icon-512x512.png - 512x512ピクセルサイズのPNGアイコン

これらのアイコンは万博2025のテーマカラー（青・ピンク系）を基調としたデザインが望ましいです。
画像編集ソフトで作成するか、既存のロゴをこのサイズに合わせて保存してください。

万博のロゴカラーは以下を参考にしています：
- 青: rgb(0, 104, 183)
- ピンク: rgb(255, 100, 161)
- 背景: rgb(255, 245, 248)
